Origem do HTML (voltar)

O HTML foi criado por Tim Berners-Lee em 1991.

HTML não é uma linguagem de programação, é uma linguagem de marcação que define a estrutura de uma página web.

A sigla significa HyperText Markup Language ou em português Linguagem de Marcação de Hipertexto.


O que são tags (voltar)

As Tags são elementos que compõem a estrutura de um documento HTML.

Elas são delimitadas por caracteres especiais (<Tag></Tag>) e são utilizadas para definir a estrutura e o conteúdo de uma página web.


O que são atributos (voltar)

Os atributos são utilizados para fornecer informações adicionais sobre um elemento HTML.

Eles são definidos dentro da tag de abertura de um elemento e são compostos por um nome e um valor (<Tag nome="valor">).


Exemplos de tags (voltar)

Tags de texto

As tags de texto são utilizadas para exibir conteúdo textual na página web.

  1. <h1> -Tag de título

  2. <h2> -Tag de título

  3. <h3> -Tag de título

  4. <h4> -Tag de título

  5. <h5> -Tag de título
  6. <h6> -Tag de título

  7. <p> - Tag de parágrafo

  8. <blockquote> - Tag de citação

Tags de links

As tags de links são utilizadas para criar links para outras páginas web.

  1. <a> - Tag de link

Tags de formatação

As tags de formatação são utilizadas para formatar o texto na página web.

  1. <strong> - Tag de texto semanticamente intenso e em negrito
  2. <b> - Tag de texto em negrito
  3. <em> - Tag de ênfase
  4. <i> - Tag de texto em itálico
  5. <mark> - Tag de texto destacado
  6. <u> - Tag de texto sublinhado
  7. <s> - Tag de texto riscado
  8. <code> - Tag de texto em código

Tags de lista

As tags de lista são utilizadas para criar listas de itens na página web.

  1. <ul> - Tag de lista não ordenada
  2. <ol> - Tag de lista ordenada
  3. <li> - Tag de item de lista


Exemplos de atributos (voltar)

Atributos de links

Os atributos de links são utilizados para fornecer informações adicionais sobre um link.

  1. href - Atributo de site de destino
  2. target - Atributo de modo de destino

Atributos de cores

Os atributos de cores são utilizados para fornecer informações adicionais sobre a cor de um elemento.

  1. color - Atributo de cor do texto

  2. background-color - Atributo de cor de fundo


Outros atributos

Outros atributos são utilizados para fornecer informações adicionais sobre um elemento.

  1. id - Atributo de identificação
  2. class - Atributo de classe
  3. style - Atributo de estilo
  4. title - Atributo de título
